# java介绍

- Java语言主要特点：
    - 跨平台，通过JVM来实现；
    - GC（垃圾自动回收）
    - 与C/C++相比，完全面向对象，避免了C/C++之中复杂的指针关系，而使用了更为简单的引用方式来进行内存传递；
    - 多线程，安全性高；

- JDK的组成
    - JDK:JAVA开发工具包，即JRE+JAVA开发工具(如javac编译工具，java运行工具,jar打包工具)
    - JRE:JAVA开发环境，即JVM(JAVA虚拟机)+核心类库
